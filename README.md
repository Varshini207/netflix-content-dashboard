# Interactive Netflix Content Dashboard 🎬

An end-to-end web application visualizing Netflix's global content library.

### 🚀 Tech Stack
- **Frontend:** React.js, D3.js (Data Visualization), Axios
- **Backend:** Python, Flask, Pandas
- **Dataset:** Netflix Titles from Kaggle

### 📊 Features
- Interactive Bar Charts showing top genres.
- Data cleaning and preprocessing via Python.
- Real-time API communication between Flask and React.

### 🛠️ Setup
1. Clone the repo.
2. Run `python app.py` in the `/backend` folder.
3. Run `npm start` in the `/frontend` folder.
